JVM

JVM(Java Virtual Machine)Java虚拟机是Java实现“一次编写、到处运行”的基本平台。他的总体结构是按照下图所示。

一个java的程序执行大体流程

  1. 通过编译java源文件成字节码文件、或者动态生成的Java字节码文件
  2. 将字节码文件（.class文件）通过类加载->验证->准备->解析->初始化等过程将类以及变量或者方法信息加入到JVM内存中。
  3. 运行时数据区分为了5个部分分别存放不同类型的数据
  4. 执行引擎通过给定的指令以及分配的处理机按照一定顺序执行程序
  5. 将类卸载、程序结束。
