Redis Database

Redis 基础 Redis是由意大利人Salvatore Sanfilippo（网名Antirez）开发。名字是由"Remote Dictionary Service”（远程字典服务）。 安装Redis Docker # 拉取Redis镜像 > docker pull redis # 运行Redis容器 > docker run --name myredis -d -p 6379:6379 redis # 执行容器中的 redis-cli, 可以直接使用命令行操作redis > docker exec -it myredis redis-cli MacOS > brew install redis Ubuntu > apt-get install redis RedHat > yum install redis 基本数据结构 Redis中最基本的数据结构有5种，分别是string、list、hash、set、zset。 其中所有的结构都是Key-Value的形式存在的，以上的五种数据结构指的都是Value的值。其中Key均为字符串类型。 string 字符串的具体结构留到以后来分析，其主要的操作如下 # 键值对操作 > set name bb OK > get name "bb" > exists name (integer) 1 > del name (integer) 1 > get name (nil) # 批量键值对操作 > mset name1 bb name2 yy OK > mget name1 name2 1) "bb" 2) "yy" > mget name1 name2 name3 1) "bb" 2) "yy" 3) (nil) # 处理过期、以及set命令的扩展 > set name bb OK > expire name 5 (integer) 1 .
